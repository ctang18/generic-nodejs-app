# generic-nodejs-app
Generic NodeJS, Express, MongoDB setup for quick development.

## Frameworks
- Express
- Mongoose
- Passport
- Jsonwebtoken

## Installation
```
npm install
```
Configure /server/config.json to your liking.

## Running the Server
```
npm start
```

The client will then be accessible at `http://localhost:3001`.
